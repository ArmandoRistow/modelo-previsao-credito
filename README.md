# 🤖 Modelo de Previsão de Crédito

## 📌 Descrição
Projeto de Inteligência Artificial para prever a aprovação de crédito de clientes com base em suas características.

## 🚀 Tecnologias
- Python
- Scikit-learn

## 📊 Objetivo
Criar um modelo preditivo para auxiliar na decisão de aprovação de crédito.

## 🎯 Resultado
Modelo com aproximadamente 86% de precisão.
